1.在List添加元素：

```c#
List<string> myList = new List<string> { "元素2", "元素3" };
myList.Insert(0, "元素1");  // 在索引0处插入 "元素1"

// 现在 myList 包含了 ["元素1", "元素2", "元素3"]
```

Add是直接添加到最后

```c#
List<string> myList = new List<string>();
myList.Add("元素1");
myList.Add("元素2");
// 现在 myList 包含 ["元素1", "元素2"]

```
